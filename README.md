# 本日の予定ビューア

Mac の Calendar.app に表示されている全カレンダーから、**今日の予定を時間順に取得して表示する**ローカルスクリプト。

`.command` をダブルクリックすると、こんな形で出力されます。

```
おはようございます

【本日の予定】
・スケジュール確認 プラネットくん
・全体会議
```

- OAuth / API キー不要（Calendar.app から直接読む）
- 実行はローカルのみ。外部サーバーへの送信なし
- 予定がない日は `・本日の予定はありません` と表示
- 実行後、出力が**クリップボードに自動コピー**される

---

## ダウンロード

1. このページ右上の緑の「**Code**」ボタンをクリック
2. 「**Download ZIP**」を選択
3. ダウンロードした ZIP を解凍して、好きな場所に置く（例: `~/Documents/DailyScheduleViewer/`）

---

## セットアップ（初回のみ）

### 1. ランチャーに実行権限を付ける

ターミナルを開いて、以下を実行してください。

```bash
chmod +x ~/Documents/DailyScheduleViewer/本日の予定.command
```

> フォルダの場所を変えた場合はパスを読み替えてください。

### 2. ダブルクリックで起動

`本日の予定.command` をダブルクリックします。

- **初回だけ** 専用の Python 環境を自動作成します（数十秒〜1 分ほど）。手動の `pip install` は不要です。
- 続けて macOS が「**カレンダーへのアクセス**」を確認するので **「OK」で許可**してください。

以降はダブルクリックするだけで即表示されます。

---

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `today_schedule.py` | 本体（カレンダー取得・整形ロジック） |
| `本日の予定.command` | ダブルクリック用ランチャー |
| `README.md` | このファイル |

Python 環境（venv）は `~/.venvs/daily-schedule-viewer/` に自動作成されます。フォルダ内には作成されないので、iCloud 同期などに影響しません。

---

## カスタマイズ

`today_schedule.py` の冒頭にある定数を編集することで挙動を変えられます。

| 定数 | 既定 | 説明 |
| --- | --- | --- |
| `GREETING` | `おはようございます` | 先頭の挨拶文 |
| `HEADER` | `【本日の予定】` | 見出し |
| `NO_EVENTS_LINE` | `・本日の予定はありません` | 予定ゼロ時の表示 |
| `SHOW_TIME` | `True` | `False` にすると時刻表示をなくす |

## トラブルシュート

| 症状 | 対処 |
| --- | --- |
| `カレンダーへのアクセスが許可されていません` | システム設定 > プライバシーとセキュリティ > カレンダー で、ターミナルを許可 |
| ダブルクリックしても開かない | `chmod +x "本日の予定.command"` を実行 |
| 初回セットアップが失敗する | ネット接続を確認 → `rm -rf ~/.venvs/daily-schedule-viewer` してから再度ダブルクリック |
| 予定が一部出ない | Calendar.app でそのカレンダーの表示チェックがオンか確認 |
